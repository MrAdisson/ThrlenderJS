## About

This project is a simple Blender inspired project built with Three.js and TypeScript.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone this repo
   ```sh
   git clone https://github.com/MrAdisson/ThrlenderJS.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
